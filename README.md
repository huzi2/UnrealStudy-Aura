## 프로젝트 개요

- 언리얼 5.3 C++과 GAS를 활용한 쿼터뷰 RPG 게임
  
## 제작 기간

- 24.02.08 ~ (개발 중)

## 개발 환경

- Unreal Engine 5.3
- C++
- 게임플레이 어빌리티 시스템(Gameplay Ability System)
- IDE : Visual Studio 2022

## 사용 기술

### 게임플레이 어빌리티 시스템
- 어빌리티 시스템 컴포넌트
- 어트리뷰트 세트
- 게임플레이 이펙트

### 게임 로직
- 마우스에 닿은 적 강조
- 향상된 입력

### 프로그래밍
- 인터페이스

### UI
- 구형 체력, 마나 표시
